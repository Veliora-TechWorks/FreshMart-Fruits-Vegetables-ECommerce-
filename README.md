# 🌿 FreshMart - Premium Organic E-Commerce Platform

A modern, responsive e-commerce platform for fresh fruits and vegetables built with Next.js 14, TypeScript, and Tailwind CSS.

## ✨ Features

- **Modern Design**: Clean, organic-inspired UI with soft shadows and rounded elements
- **Responsive Layout**: Mobile-first design that works on all devices
- **Product Showcase**: Featured products with add-to-cart functionality
- **Interactive Components**: Customer reviews carousel, smooth animations
- **Brand Identity**: Fresh green and warm yellow color scheme
- **Performance Optimized**: Built with Next.js 14 for optimal performance

## 🎨 Design System

### Colors
- **Primary Green**: #80B500
- **Accent Mint**: #F2F9E6
- **Headings**: #222222
- **Body Text**: #666666
- **Background**: #FFFFFF
- **Accent Yellow**: #FFD700

### Typography
- **Font**: Inter (Google Fonts)
- **Headings**: Bold, large sizes
- **Body**: Light, readable text

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser
